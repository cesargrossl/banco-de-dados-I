# Álgebra Relacional

Disciplina de Banco de Dados I - Engenharia de Software

---

# Sobre

A Álgebra Relacional é uma linguagem formal utilizada em Banco de Dados Relacionais para realizar consultas e manipulação de dados através de operadores matemáticos.

Ela é considerada a base teórica do SQL.

- Divisão
